# tenholes-blues-down
Java下载蓝调口琴网（http://www.tenholes.com）蓝调口琴琴谱、伴奏资源

<br>

<br>

尊重版权，使用这个爬虫需要自己有会员权限的账号!!

<br>

<br>

## 功能

### 已实现

1.蓝调口琴谱图片下载保存

2.蓝调口琴伴奏下载保存

<br>

## 使用方法

1.登录蓝调口琴网（http://www.tenholes.com/）复制Cookie

2.在userConfig.properties文件中填入Cookie

3.运行SpiderMan.java
